## m3u8 资源探测器

编写一个Chrome扩展，能够在用户访问网页时自动检测页面中是否包含m3u8资源链接。如果检测到m3u8链接，扩展应在浏览器工具栏显示一个图标，点击该图标后弹出一个窗口，列出所有检测到的m3u8链接，并提供复制链接的功能。后续可以考虑调用外部程序进行下载，比如通过websocket与本地的下载工具通信。


### 功能需求
1. **自动检测m3u8链接**：当用户访问网页时，扩展应自动扫描页面内容，查找所有包含m3u8资源的链接。
2. **流量监控**：扩展应能够监控网络请求，捕获所有m3u8资源的请求。
3. **图标显示**：当检测到m3u8链接时，扩展应在浏览器工具栏显示一个图标，提示用户有可用的m3u8资源。
4. **弹出窗口**：点击工具栏图标后，弹出一个窗口，列出所有检测到的m3u8链接。
5. **复制链接功能**：在弹出窗口中，用户应能够方便地复制m3u8链接到剪贴板。
6. **用户界面**：弹出窗口应具有简洁直观的用户界面，方便用户查看和操作m3u8链接。
7. **扩展设置**：提供基本的扩展设置选项，如启用/禁用自动检测功能。
8. **调用外部程序**：后续可以考虑通过websocket与本地的下载工具通信，实现m3u8资源的下载功能。
### 技术需求
1. **Chrome扩展开发**：使用Chrome扩展API进行开发，确保扩展能够在Chrome浏览器中正常运行。
2. **内容脚本**：使用内容脚本扫描网页内容，检测m3u8链接。
3. **网络请求监听**：使用Chrome的webRequest API监听网络请求，捕获m3u8资源请求。
4. **消息传递**：使用Chrome扩展的消息传递机制，在内容脚本和后台脚本之间传递检测到的m3u8链接。
5. **用户界面开发**：使用HTML、CSS和JavaScript开发弹出窗口的用户界面。
6. **剪贴板操作**：使用Clipboard API实现复制链接到剪贴板的功能。
7. **存储管理**：使用Chrome的storage API管理扩展的设置和用户偏好。


### 测试
1. **功能测试**：确保扩展能够正确检测m3u8链接，并在工具栏显示图标。
2. **用户界面测试**：确保弹出窗口的用户界面简洁直观，复制链接功能正常工作。
3. **兼容性测试**：确保扩展在不同版本的Chrome浏览器中均能正常运行。
4. **性能测试**：确保扩展在检测m3u8链接时不会显著影响网页加载速度和浏览器性能。
5. **边界情况测试**：测试扩展在没有m3u8链接的网页上运行时的表现，确保不会误报或崩溃。
