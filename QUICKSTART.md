# 快速开始指南

## 🚀 5分钟快速安装

### 步骤1：确认Chrome版本
确保您的Chrome浏览器版本 >= 88（支持Manifest V3）

### 步骤2：下载扩展
```bash
# 如果您有git
git clone https://github.com/yourusername/m3u8-detector.git

# 或者直接下载ZIP文件并解压
```

### 步骤3：加载到Chrome

1. 打开Chrome，在地址栏输入：
   ```
   chrome://extensions/
   ```

2. 在右上角开启"开发者模式"

3. 点击"加载已解压的扩展程序"

4. 选择项目文件夹 `m3u8-detector`

5. 完成！扩展图标会出现在工具栏

## 💡 使用示例

### 示例1：检测视频网站

1. 访问任何视频网站（如优酷、爱奇艺、腾讯视频等）
2. 播放视频
3. 观察扩展图标 - 如果出现数字Badge，说明检测到m3u8资源
4. 点击图标查看所有检测到的链接
5. 点击"复制"按钮获取链接

### 示例2：扫描整个页面

即使视频还未播放，扩展也会自动扫描页面：
- 检查所有链接
- 分析JavaScript代码
- 监听后续的网络请求

### 示例3：自定义设置

1. 点击扩展图标
2. 点击"设置"按钮
3. 根据需要开启或关闭功能：
   - **自动检测**：是否扫描页面DOM
   - **显示通知**：检测到资源时是否通知（待实现）

## 🔧 常见问题

### Q: 为什么检测不到资源？
A: 可能的原因：
- 网站使用了加密或混淆技术
- 视频尚未开始加载
- 网站不使用m3u8格式（可能用其他格式如MP4、DASH等）

### Q: 如何更新扩展？
A: 
1. 前往 `chrome://extensions/`
2. 找到"M3U8资源探测器"
3. 点击刷新按钮 🔄

### Q: 检测到的链接无法播放？
A: 
- 某些m3u8链接可能有时效性或需要特定的header
- 建议使用专业的m3u8播放器或下载工具

### Q: 如何导出检测到的链接？
A: 
- 目前需要手动复制
- 导出功能正在开发中
